# VSCode.Pascal
Setup for Compiling Pascal projects

# Extension:
## Pascal, by Alessandro Fragnani
Home: https://marketplace.visualstudio.com/items?itemName=alefragnani.pascal
- source: https://github.com/alefragnani/vscode-language-pascal
- install: (Ctrl+P) `ext install alefragnani.pascal`

## Pascal Formatter
Home: https://marketplace.visualstudio.com/items?itemName=alefragnani.pascal-formatter
- install: (Ctrl+P) `ext install alefragnani.pascal-formatter`

## Free Pascal (fpc) Debugger
Home: https://marketplace.visualstudio.com/items?itemName=CNOC.fpdebug
- install: (Ctrl+P) `ext install CNOC.fpdebug`
